# University Registration System

This project is a terminal-based university registration system developed using Python.

## Features
- View available courses
- Register for courses
- View student schedule
- Calculate tuition bill
- Error handling

## Concepts Used
- Classes and Objects
- Constructors
- Encapsulation
- Getter and Setter
- Inheritance
- Polymorphism
